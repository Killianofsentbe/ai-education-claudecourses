📚 학습 방법
권장 학습 순서와 각 단계에서 활용할 수 있는 기능을 안내합니다.
01

메인 페이지에서 강의 선택

메인 페이지에서 관심 있는 강의 카드를 클릭합니다. 카드는 AI Fluency 시리즈 · API 개발 · MCP·Claude Code 3개 섹션으로 분류되어 있습니다. 📹 영상 뱃지가 있는 강의는 영상 강의입니다.

02

강의 페이지에서 레슨 학습

좌측 사이드바의 레슨 목록을 클릭하거나 상단 탭을 이용해 레슨을 이동합니다. 각 레슨은 학습 목표 → 핵심 내용 → 요약 구조로 구성되어 있습니다.

03

NotebookLM으로 심층 학습

각 강의 페이지 상단의 ⬡ NotebookLM 열기 버튼을 클릭합니다. 강의 자료 전체가 탑재된 NotebookLM에서 AI와 대화하며 궁금한 내용을 질문하거나 요약을 요청할 수 있습니다.

04

영상 강의 병행 (선택)

영상이 포함된 6개 강의(01·04·05·08·09·10)는 원본 Anthropic Academy 사이트에서 영상을 함께 시청할 수 있습니다. 텍스트 자료와 병행하면 학습 효율이 높아집니다.
