강의 페이지

각 강의의 레슨 전체 내용을 좌측 사이드바 클릭으로 탐색. 상단 배너에 원본 영상·NotebookLM 버튼, 레슨마다 개별 영상 링크 제공.

예시: 01번 강의 →
📚 학습 방법
권장 학습 순서와 각 단계에서 활용할 수 있는 기능을 안내합니다.
01

메인 페이지에서 강의 선택

메인 페이지에서 관심 있는 강의 카드를 클릭합니다. 카드는 AI Fluency 시리즈 · API 개발 · MCP·Claude Code 3개 섹션으로 분류되어 있습니다. 📹 영상 뱃지가 있는 강의는 영상 강의입니다.

02

강의 페이지에서 레슨 학습

좌측 사이드바의 레슨 목록을 클릭해 레슨을 이동합니다. 각 레슨은 학습 목표 → 핵심 내용 → 요약 구조로 구성되어 있습니다.

03

NotebookLM으로 심층 학습

각 강의 페이지 상단 배너의 📺 원본 영상 → 버튼으로 Anthropic Academy 원본 영상을, ⬡ NotebookLM → 버튼으로 AI 심화 학습을 바로 연결할 수 있습니다. 레슨별 영상 링크도 각 레슨 헤더에 개별 제공됩니다.

04

영상 강의 병행 (선택)

영상이 포함된 6개 강의(01·05·06·09·10·11)는 원본 Anthropic Academy 사이트에서 영상을 함께 시청할 수 있습니다. 텍스트 자료와 병행하면 학습 효율이 높아집니다.
