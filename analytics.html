Analytics 대시보드 설정

Claude Courses 사이트의 조회수와 사용자 현황을 확인할 수 있어요.
최초 1회만 설정하면 됩니다.

숫자로만 이루어진 ID예요. Google Analytics → 관리 → 속성 설정 → 속성 ID에서 확인하세요.
Google Cloud Console → 사용자 인증 정보에서 생성하세요. 아래 안내를 참고해주세요.

OAuth2 Client ID 만들기

  1. console.cloud.google.com 접속 → 프로젝트 선택 또는 새로 만들기
  2. API 및 서비스 → 라이브러리 → Google Analytics Data API 검색 → 사용 설정
  3. API 및 서비스 → 사용자 인증 정보 → + 사용자 인증 정보 만들기 → OAuth 클라이언트 ID
  4. 애플리케이션 유형: 웹 애플리케이션
  5. 승인된 JavaScript 원본에 추가:
    https://killianofsentbe.github.io
    http://localhost:8080 (로컬 테스트용)
  6. 만들기 → 클라이언트 ID 복사 → 위에 붙여넣기
